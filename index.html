This is a memory based card game that I built with HTML, CSS, and JavaScript while I was at Mindweaver. The goal of the game is to find two matching cards within the same turn in order to register it as a point. If this is accomplished, the cards would disappear from the game board and a point would be assigned to the player. If not, the non-matching cards would turn back over and your turn would end. This small project taught me how to incorporate certain techniques accomplish this, and I have since used those skills to further improve my others works.
